### Tilde 
- `~2.1`, means 2.1.0 <= version < 3.0.0
- `~2.1.1`, means 2.1.1 <= version < 2.2.0

### Caret
// - `^2.1`, means 2.1.0 <= version < 3.0.0   
- `^2.1.1`, means 2.1.0 <= version < 3.0.0   

### Wildcard
- `2.1.*`, means 2.1.0 <= version < 2.2.0


[reference: composer](https://getcomposer.org/doc/articles/versions.md)
